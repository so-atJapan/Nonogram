## 機能一覧
・サインアップ<br>
・ログイン<br>
・PLAY <br>
・CREATE<br>
・EXIT<br>
・メニューバー

### 🔳サインアップ
nonogramをプレイするにあたってアカウントの作成が可能

### 🔳ログイン
アカウントの情報がDB上で管理され、IDとパスワードを用いてログインができる<br>


### 🔳PLAYボタン
問題(難易度)一覧 <br>
EASY,NORMAL,HARD<br>

### 🔳CREATEボタン
独自で縦,横のマス数の設定が可能　<br>
塗りつぶすマス数の数字の設定が可能

### 🔳EXITボタン
ページを閉じる。メニュー → 終了でも同様の操作が可能

### 🔳メニューバー
・ホーム<br>
・同期<br>
・ヘルプ<br>
・終了<br>

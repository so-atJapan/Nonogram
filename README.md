# Nonogram 
世界的に有名なピクロスパズルゲーム

## 遊び方
１．赤枠で囲んだ数字をもとにマスを塗りつぶす。例1は解答例。<br>
(例1)<br>
![example1](/imageReadme/example1.png) <br>
数字は連続で塗りつぶせるマスの数を表す。縦マスも同様。<br>


## 機能一覧
・サインアップ<br>
・ログイン<br>
・PLAY <br>
・CREATE<br>
・メニューバー<br>
・EXIT
